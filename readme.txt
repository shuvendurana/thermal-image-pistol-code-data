Title: A Two-Stage Deep Learning Framework for Concealed Pistol Detection in Thermal Images Using Background Suppression



**‎Description – An overview of the code/dataset**.



The dataset used in this study consists of thermal images collected from both existing sources and a self-created dataset. Initially, 358 thermal images were obtained from a publicly available concealed pistol detection dataset. However, due to the limited size, additional data were generated by creating a simulated pistol structure to mimic real-world concealed weapon scenarios under controlled conditions .



The final dataset comprises 1,449 thermal images, including 1,048 images with pistols and 401 images without pistols, ensuring a balanced representation of both classes . These images were captured using a thermal camera, which records heat signatures to highlight temperature differences between the human body and concealed metallic objects.



For model development, the dataset was divided into three subsets: training (800 images), validation (448 images), and testing (221 images). The images include variations in human subjects, clothing conditions, and concealment positions, making the dataset robust for real-world surveillance scenarios.



Overall, the dataset plays a crucial role in enabling accurate classification and detection of concealed pistols, while maintaining safety and ethical considerations during data collection.



**‎Code Information:** 

the code involves three sections.

1. ViT classification: The code implements a thermal image classification system using a Vision Transformer (ViT) model. Images are preprocessed and loaded using PyTorch DataLoaders after organizing them into training, validation, and testing sets. A pre-trained ViT model is fine-tuned for binary classification (with pistol and without pistol). The model is trained using the Adam optimizer and Cross-Entropy loss function. Performance is evaluated using accuracy, precision, recall, AUC, and confusion matrix. Finally, the trained model is tested to assess its effectiveness on unseen data.
for this refer: vision\_transformer classification.ipynb file
2. YOLOV8 Object Detection with background: After classification, for the classified thermal images with pistol, code implements concealed pistol detection using the YOLOv8 model with background. Thermal images with pistols are used to train the model for detecting pistols. During training, the model learns to localize pistols using bounding boxes. Performance is evaluated using metrics like precision, recall, and mAP. Finally, the model detects and highlights pistols in test images.
for this refer: gundetectionwithbackgrund.ipynb file
3. YOLOV8 Object Detection without background: By using thermal images with background, we observed that we got more false positives from the background. Then code implements concealed pistol detection using YOLOv8 with background removal on thermal images. Initially, YOLOv8 segmentation is applied to remove the background and retain only the human region. The processed images are then used to train a YOLOv8 object detection model for identifying pistols. The model learns to accurately localize pistols using bounding boxes. Performance is evaluated using metrics such as precision, recall, and mAP. This approach improves detection accuracy by reducing background noise.

&#x20;  for this refer: gunwithbackremoval.ipynb file

**‎Methodology (if applicable) – Steps taken for data processing or modeling.**



The proposed method uses a two-stage deep learning framework for concealed pistol detection in thermal images. First, images are preprocessed and classified using a Vision Transformer (ViT) into pistol and non-pistol categories. Next, YOLOv8 segmentation is applied to remove the background from images containing pistols. The refined images are then passed to the YOLOv8 object detection model to localize pistols using bounding boxes. Finally, the system is evaluated using metrics such as accuracy, precision, recall, F1-score, and mAP.
